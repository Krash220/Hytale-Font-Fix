# Hytale Font Fix Mod

This plugin modifies the game's font texture size limit by raising it to 10240x10240, and includes a font which includes 42,247 characters (CJK).

![preview](https://github.com/user-attachments/assets/493bbeae-76e6-4e92-8296-5fbaef7801a4)

## How to Install:
1. Download the mod from Release.
2. Open the Hytale client, click the gear icon in the upper-left corner, then click "Open directory".
3. Navigate to `install\release\package\game\latest\Client`.
4. Extract the plugin into this folder. At this point, version.dll should be located in the same directory as HytaleClient.exe.
5. Enjoy it!
